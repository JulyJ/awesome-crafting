**Step-by-Step Slicer Guide:**
1. Upload your STL file (free models from Thingiverse, MyMiniFactory)
2. Set slice thickness (match your material: 1/8", 1/4", etc.)

4. Download SVG layers
5. Cut each layer on your laser cutter
6. Stack and glue layers together
7. Sand and finish as desired

## Advanced Techniques

### Layered Map Creation Guide

Creating stunning topographical maps using the Laser Slicer tool:

**Materials Needed:**
- STL file of terrain (download from USGS, OpenTopography)
- 1/8" material (wood or cardboard) - 5-15 sheets depending on elevation
- Wood glue or acrylic cement
- Stain or paint (optional)
- Sealer/finish

**Process:**
1. **Find terrain data**: Download from OpenTopography.org or USGS Earth Explorer
2. **Upload to Laser Slicer**: Set slice thickness to match material
3. **Adjust parameters**: Set number of layers, spacing, base size
4. **Download all layers**: Each layer is a separate SVG
5. **Cut in sequence**: Label each piece by layer number
6. **Stack and glue**: Start with base, work upward
7. **Finishing**: Sand edges, stain/paint, seal

**Popular Map Subjects:**
- National parks (Grand Canyon, Yosemite)
- Local hometown areas
- Famous mountain ranges
- Island coastlines
- Lake bathymetry
